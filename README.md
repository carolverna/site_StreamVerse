## Sobre
StreamVerse é um site/projeto que ajuda a encontrar os melhores filmes e séries nas principais plataformas de streaming: Netflix, Amazon Prime Video e Disney+.

## Tecnologias usadas
- HTML5
- CSS3
- JavaScript
